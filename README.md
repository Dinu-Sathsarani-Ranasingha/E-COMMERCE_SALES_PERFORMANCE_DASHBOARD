# E-COMMERCE_SALES_PERFORMANCE_DASHBOARD

Python | Pandas | SQL | SQLite | Power BI | Excel | GitHub

Conducted end-to-end analysis of an e-commerce sales dataset, including data cleaning, exploratory data analysis (EDA), SQL querying, KPI calculation, and interactive dashboard development.
Built a Power BI dashboard featuring revenue, profit, regional performance, category analysis, product profitability, and time-based sales trends.
Performed business analysis using Python and SQL to identify sales patterns, profit drivers, and loss-making products, providing actionable insights.
Documented the complete project workflow, code, dashboard, and business insights on GitHub.
